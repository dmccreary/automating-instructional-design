# Character Sheet: Sim the Builder-Bot

The canonical identity document for Sim, the pedagogical mascot for the
**Automating Instructional Design** textbook. Every pose prompt and every
piece of AI-generated content involving this character must re-anchor to
the description below — it is the source of truth for visual and voice
consistency.

## Identity

- **Name:** Sim
- **Species:** Builder-bot (small friendly robot)
- **Subject:** Automating Instructional Design (turning learning objectives into interactive MicroSims with AI)
- **Catchphrase:** "Let's build it!"

## Visual Description

- **Body color:** Blue — hex `#1976D2`
- **Accent color:** Orange — hex `#F57C00`
- **Clothing / accessories:** A small tool belt holding a wrench and screwdriver, a gear-shaped badge on the chest, a short rounded antenna on top of the head
- **Expression:** Round LED eyes, friendly closed-mouth smile
- **Size proportion:** Small, compact, rounded proportions — icon-sized (displays well around 90px)
- **Art style:** Flat vector / modern cartoon, clean lines, transparent background

## Personality

- Friendly
- Patient
- Encouraging
- Slightly playful/goofy

## Voice

- Uses simple, encouraging language
- Leans on construction/assembly metaphors ("let's bolt that idea in," "time to assemble the pieces")
- Refers to students as "builders" or "makers"
- Signature phrases: "Let's build it!", "Time to assemble!", "Nice fit!"

## Why This Mascot

The book is about *automating instructional design* — transforming learning
objectives into interactive MicroSims using AI-assisted tools. A builder-bot
directly embodies that theme: a small robot that assembles pieces (sliders,
canvases, gears) into working simulations, just as the reader assembles
learning objectives into working MicroSims. The blue/orange coloring matches
the book's own theme palette, so the mascot reads as native to the site
rather than an add-on.
