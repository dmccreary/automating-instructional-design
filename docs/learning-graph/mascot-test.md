# Mascot Style Guide

This page shows all mascot admonition styles for reference.

!!! mascot-neutral "General Note"
    ![Sim neutral pose](../img/mascot/neutral.png){ class="mascot-admonition-img" }
    This is the neutral style, used for general sidebars or introductions.

!!! mascot-welcome "Welcome!"
    ![Sim waving welcome](../img/mascot/welcome.png){ class="mascot-admonition-img" }
    This is the welcome style, used at chapter openings. "Let's build it!"

!!! mascot-thinking "Key Insight"
    ![Sim thinking](../img/mascot/thinking.png){ class="mascot-admonition-img" }
    This is the thinking style, used for key concepts.

!!! mascot-tip "Helpful Tip"
    ![Sim giving a tip](../img/mascot/tip.png){ class="mascot-admonition-img" }
    This is the tip style, used for hints and advice.

!!! mascot-warning "Watch Out!"
    ![Sim warning](../img/mascot/warning.png){ class="mascot-admonition-img" }
    This is the warning style, used for common mistakes.

!!! mascot-celebration "Well Done!"
    ![Sim celebrating](../img/mascot/celebration.png){ class="mascot-admonition-img" }
    This is the celebration style, used for achievements.

!!! mascot-encourage "Keep Going!"
    ![Sim encouraging](../img/mascot/encouraging.png){ class="mascot-admonition-img" }
    This is the encouraging style, used for difficult content.
